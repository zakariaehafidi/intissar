## Deploying to GitHub Pages

To host this website on GitHub Pages:

1. Create a new repository named `intissar-site` on GitHub
2. Push these files to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/intissar-site.git
   git push -u origin main
   ```

3. Go to your repository settings on GitHub
4. Navigate to the "Pages" section
5. Under "Source", select the `main` branch
6. Click "Save"

Your site will be published at: `https://YOUR_USERNAME.github.io/intissar-site/`

## Customization

To customize the site:
- Edit `index.html` to change content
- Edit `style.css` to change colors, fonts, or layout
- Replace the description text in the `<p>` tag within the `.description` section
